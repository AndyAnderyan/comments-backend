# Модуль «Коментарі» — проєктна специфікація

Ця папка містить узгоджену архітектуру фронтенду (Angular) та контрактів бекенду (NestJS) для модуля коментарів до об'єктів на мапі.

Вміст:
- frontend-architecture.md — структура модулів, компонентів, сервісів, роутів, стан (NgRx)
- api/rest.md — REST-контракти, фільтри, сортування, пошук, пагінація
- api/openapi.yaml — OpenAPI-специфікація REST API
- ws-events.md — події WebSocket/внутрішні нотифікації
- data-models.md — моделі даних, узгодження з PostgreSQL/ORM
- permissions-rbac.md — права доступу та правила гардів
- ui-specs.md — вимоги до інтерфейсів (загальний та картка об’єкта)
- performance.md — продуктивність, кешування, пагінація
- acceptance-criteria.md — критерії приймання, сценарії користувача

Нотації:
- ID елемента мапи: пара `layerTypeId` + `objectId`.
- Відповіді: вкладеність до 3 рівнів (thread-like).
- Дані — JSON через REST, push-оновлення через WS.
