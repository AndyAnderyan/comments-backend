# UI/UX специфікація

## Загальний інтерфейс «Коментарі» (/comments)
- Ліва панель: список об’єктів із коментарями (тип, назва, остання активність), сортування
- Верхня панель: пошук (текст/Автор/ID об’єкта), фільтри
- Права область: треди обраного об’єкта; pinned зверху; хронологія; індикатори непрочитаного
- Дії:
  - Створити новий коментар
  - Відповісти біля кожного коментаря/відповіді (до рівня 3)
  - Редагувати/Приховати (Автор, Admin), Видалити (Admin)
  - Pin/Unpin (Автор, Admin, якщо інший не pinned)
- Перемикач «Лише коментарі без відповідей» (onlyRoots)

## Картка об’єкта (/comments/object/:layerTypeId/:objectId)
- Список коментарів цього об’єкта, сортування за часом створення; pinned зверху
- Ті самі дії і індикатори, що й у загальному інтерфейсі
- Кнопка переходу до повного списку

## Стани та доступність елементів
- На рівні 3 — кнопка «Відповісти» неактивна
- При наявності прихованих коментарів — відмітка «Приховано» (видима тільки Admin)
- Непрочитані — виділення кольором/іконкою до позначення як прочитано

## Валідації
- Текст до 2000 символів, обов’язковий
- Не дозволяти створити другий pinned для того самого об’єкта

## Повідомлення
- Підтвердження при видаленні/приховуванні треду (каскадний ефект)
